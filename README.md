# Basic Banking System (Java)

## Description
A simple console-based banking system implemented in Java.  
Supports account creation, login, deposits, withdrawals, and balance checking.

## Features
- Create and login to accounts
- Deposit and withdraw money
- Check balance
- Basic input validation and exception handling

## Concepts Used
- Object-Oriented Programming (OOP)
    - Encapsulation
    - Constructors & parameterized constructors
    - Access modifiers
- Exception Handling (`InputMismatchException`)
- Control flow (`if-else`, loops, `switch-case`)
- Arrays for in-memory storage
